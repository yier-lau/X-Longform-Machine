# Cover Prompt

Use this when generating a conceptual cover prompt for an X longform article.

## Selection

Extract one core visual phrase from the article.

Prefer 2 to 8 Chinese characters. Use the article title only if it is short and visually strong.

Good cover text should be:

- direct
- visual
- a little strange
- connected to the article's strongest line

## Prompt Template

Generate a 5:2 advanced conceptual poster for an X longform cover, not a normal illustration and not a simple text-on-background graphic.

Article title: `{article_title}`

Core visual phrase: `{core_text}`

Core concept: `{concept}`

Create a restrained, high-concept editorial poster that visually translates the article's main argument. The design must combine giant embedded typography, a clear horizontal supporting surface, one to three key subjects, and a strong visual metaphor.

The core text `{core_text}` must be the main visual structure, huge, clear, and integrated into the scene as architecture, wall, data structure, archive, pressure surface, or symbolic field. It must not float as pasted text.

Use a clear supporting plane in the lower third: paper strata, desk surface, ruin platform, archive layer, memory floor, or simplified ground. The subject must stand on or interact with this plane.

Add one small human figure or symbolic object performing a meaningful action: pulling a thread from records, rebuilding, sorting fragments, standing before the text, carrying a small archive, cutting through a wall, or extracting a signal from noise.

The typography, subject, supporting plane, and negative space must form one visual sentence. The image should be minimal but not empty, dramatic but not decorative, clever but readable.

Color system: 2 to 4 controlled colors. Prefer paper white or cold gray background, near-black typography, muted gray paper/data layers, and one tiny red or orange-red accent. Use print texture, subtle grain, silkscreen or lithograph feel. Avoid cheap gradients, neon, rainbow colors, cyberpunk, robots, cartoon style, ecommerce poster style, random numbers, fake signatures, meaningless small text, and unreadable garbled text.

If Chinese text rendering is unreliable, preserve the composition and leave a clear typographic structure area for later manual Chinese typesetting.

Mood: restrained, clear, lonely, rough, reconstructive, modern, publication-grade.

Aspect ratio: 5:2.

## Output

Output only the final image prompt unless the user asks for explanation.
